# AGENCY_47

Landing page inicial focada em conversão para o nicho **clínicas dentárias em Portugal**.

## Estrutura

- `index.html`: página principal com estrutura de conversão (hero, problema, solução, prova, oferta e CTA final).
- `styles.css`: estilos da interface, com foco em clareza, velocidade e mobile-first.

## Como visualizar

Abra o arquivo `index.html` no navegador.
