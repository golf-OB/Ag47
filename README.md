# AG47

Landing page inicial da **Agência 47** (Marketing, Desenvolvimento e Vendas) — a nascer em Portugal.

## Estrutura

- `index.html`: página principal.
- `styles.css`: estilos (tema escuro, componentes e responsividade).

## Como visualizar (local)

1. Faça download/clone do repositório.
2. Abra o ficheiro `index.html` no navegador.

> Dica: se quiser um servidor local rápido, use uma extensão como “Live Server” no VS Code.

## Checklist antes de publicar

- [ ] Substituir `https://example.com` no `canonical` e `og:url` pela URL final.
- [ ] Adicionar `og-image.jpg` (1200×630) e actualizar o caminho no `<meta property="og:image">`.
- [ ] Actualizar o email/WhatsApp para os contactos finais.

## Notas

- O formulário actual é demonstrativo (não envia dados). Para produção, ligar a um endpoint, CRM, ou integração (ex.: Zapier/Make).
